## 🔑 How GitLaunchrTKN Works

GitLaunchrTKN connects your browser session to the Bankr Agent API through GitHub login.

It does not require a wallet on the browser, unlike many crypto tools.

1. You log in with GitHub OAuth inside the app.  
2. The app sends instructions to the Bankr Agent API for launching tokens on Base mainnet.  
3. It can deploy a FeeSplitter contract on Base automatically.  
4. The app handles fees and transactions on the server side without exposing your keys.

This structure keeps your experience simple and secure.
